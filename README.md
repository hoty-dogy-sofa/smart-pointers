# Project: Умные указатели (C++)

## 📌 Описание проекта
Реализация упрощенных версий `unique_ptr` и `shared_ptr` с базовой функциональностью стандартных умных указателей.

## 🏗 Структура проекта
```bash
smart-pointers/
├── include/
│ ├── unique_ptr.hpp # Шаблон unique_ptr
│ └── shared_ptr.hpp # Шаблон shared_ptr
└── README.md
```
